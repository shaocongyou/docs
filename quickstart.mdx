## Before you start

You need an API key and your API base URL.

```bash
export AI_API_KEY="your_api_key"
export AI_BASE_URL="https://cc2.cx/v1"
```

<Info>
  Keep API keys on your server. Do not expose them in browser code, mobile apps, or public repositories.
</Info>

## 1. Generate text

Use `gpt-5.4` or `gpt-5.5` with the chat completions endpoint.

```bash
curl "$AI_BASE_URL/chat/completions" \
  -H "Authorization: Bearer $AI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "Give me 5 names for a coffee brand."}
    ],
    "temperature": 0.7
  }'
```

## 2. Generate an image

Use `image-2` with the image generation endpoint.

```bash
curl "$AI_BASE_URL/images/generations" \
  -H "Authorization: Bearer $AI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "image-2",
    "prompt": "A premium coffee cup on a marble table, warm morning light",
    "size": "1024x1024",
    "n": 1
  }'
```

## 3. Generate a video

Use `seedance2.0` with the video generation endpoint.

```bash
curl "$AI_BASE_URL/videos/generations" \
  -H "Authorization: Bearer $AI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance2.0",
    "prompt": "A cinematic slow push-in on a premium coffee cup with steam rising",
    "duration": 5,
    "resolution": "1080p"
  }'
```
