## Quick example

Send your API key as a Bearer token and call the endpoint you need.

```bash
curl https://cc2.cx/v1/chat/completions \
  -H "Authorization: Bearer $AI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Write a launch plan for my product."}
    ]
  }'
```

## Available models

| Model | Type | Best for |
| --- | --- | --- |
| `gpt-5.4` | Text | Fast chat, code, reasoning, tool instructions, JSON output |
| `gpt-5.5` | Text | Higher quality reasoning, planning, and more demanding writing tasks |
| `image-2` | Image | Product shots, posters, social graphics, visual concepts |
| `seedance2.0` | Video | Short clips, ad creatives, scene motion, prompt-to-video |

<Tip>
  Use `https://cc2.cx/v1` as your API base URL in OpenAI-compatible clients.
</Tip>
