# AI API Docs

Mintlify documentation for an AI API that exposes `gpt-5.4`, `image-2`, and `seedance2`.

## Development

Install the Mintlify CLI:

```bash
npm i -g mint
```

Run the docs locally from the `docs` directory:

```bash
cd docs
mint dev
```

Open `http://localhost:3000` to preview the site.

## Update before publishing

- Replace `https://api.example.com` with your real API base URL.
- Replace `support@example.com` in `docs.json` with your support email.
- Update logo files in `logo/light.svg` and `logo/dark.svg`.
